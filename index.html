Mostly Serious Study Server

Welcome to the Mostly Serious Study Server website! The purpose of this website is to complement the Mostly Serious Study Discord Server by offering convenient access to existing study materials and helpful resources. If you have any suggestions to improve this page, feel free to put them in the suggestions channel of the server!

About The Server

Server History

The Mostly Serious Study Server was intially created in 2020 by a couple of friends who wanted to share study guides with each other and to help each other with schoolwork & academics. Over time, the server has grown to include over a hundred members, and has amassed a wide catalog of both member-created and external study materials and resources to attest to this. We hope that the server will continue to grow and help more people study & succeed. The Mostly Serious Study Server currently serves the Class of 2025.

Purpose of the Server

The purpose of The Mostly Serious Study Server is to provide a convenient platform for members to help each other with school & homework. Members can share study material, arrange study sessions, and ask questions regarding recent assignments or upcoming assessments. If you have created or have access to any study material that you think will be helpful for other people, we encourage you to share it with everyone in the server.
